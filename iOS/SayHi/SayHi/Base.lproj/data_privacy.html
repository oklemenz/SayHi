Data Privacy Declaration

Every time the application "SAY Hi!" (hereinafter "App") and/or associated Services are used, data about your mobile end device are automatically collected, processed, and used. Protection of your privacy when you use the App and/or Services is important to us. Therefore, we would like to inform you below which of your personal data will be collected when you use the App and/or Services and how these data are handled.

Automated data collection

When you access the App and/or Services data are automatically sent to our server, for technical reasons. The following data are stored separately from other data that you send to us in these circumstances:
  • App Store: The store from which the App was downloaded and installed
  • App Version: The versionName (Android) or the Bundle version (iOS)
  • Country: The country the user resides in
  • Device Brand: The brand name of the mobile device
  • Device Category: The category of the mobile device (e.g. mobile or tablet)
  • Device Model: The mobile device model name
  • First Open Time: The time (in microseconds, UTC) at which the user first opened the App, rounded up to the next hour
  • Language: The language setting of the device OS
  • New/Established: First opened the App within the last 7 days
  • Established: First opened the App more than 7 days ago
  • OS Version: The version of the device OS

To use the App, neither a Google account nor any other social login is required. The App only authenticates anonymously and explicitely does not use Advertsing Identifiers (IDFA) functionality (via Google AdSupport). Therefore the following data is NOT collected automatically:
  • Age
  • Gender
  • Interest Properties

Data Released or Entered by Users

When you use the App and/or Services additional data is entered by you. The following data are stored that you send to us in these circumstances:
First App Start
The App is free of registration, but the App gives you the option to personalize your user data. You can enter the following initial information:
  • First Name
  • Gender
  • Birth Year
  • Content Language
Settings
The App gives you the option to personalize your user data in the settings. You can enter the following additional information in the settings dialog:
  • First Name
  • Gender
  • Birth Year
  • Content Language
  • Personal Status
  • Voice Recording
Profiles
The App gives you the option to add profiles. You can enter the following data:
  • Topics you like
  • Topics you don't like
  • Relation Type
The App gives you the option to add own tags and categories. You can enter the following data:
  • Tag name
  • Category name
Profile Matching
The App gives you the option to match your profiles with other users. The following data is exchanged during matching between you and the other user:
  • First name
  • Gender
  • Birth year
  • Personal status
  • Voice Recording (optional)
  • Topics you like/dislike of your active profile, depending on the chosen matching mode

Local Personal Data

The App supports encryption of all your personal user data on the device. The following personal data will never be stored as plain data on a central server:
  • First Name
  • Personal Status
  • Voice Recording
  • Profile Names

Logged Application Events

In addition to the automatically collected events by the Google Firebase platform, the following events are logged for system analysis purposes:
  • First App start
  • Creation of new profile
  • Change of profile information
  • Deletion of a profile
  • Liking of a topic
  • Disliking of a topic
  • Discard liking or disliking of a topic
  • Profile matching
  • Creation of new tag
  • Creation of new category
  • Rating of App
  • Writing of support mail
  • Recording of voice
  • Deletion of recorded voice
  • Space switch
Logged events can contain data as mentioned in the sections before. In addition the "Profile matching" event contains the location, in case location services have been allowed for the App.

Device Feature Access

To function correctly, the App needs access to the following device features:
  • Internet: Connection to application server to read and write application content data
  • Camera: Used to scan QR codes for profile matching
  • Microphone: Used to record profile matching voice (optional)
  • Location: Used to determine location during matching (optional)

Data Disclosure

Generally data is not shared on an disaggregated level with any third party. Data aggregations may be shared with third parties for marketing purposes or analytical evaluations.

By the usage of the App you agree to this data privacy declaration.
